# 短视频无水印下载小程序

一个功能完善的微信小程序，支持抖音、快手、小红书等平台短视频无水印下载，集成广告变现功能。

## 🎬 核心功能

### 视频解析功能

#### 1. 多平台支持
- **抖音视频解析** - 支持抖音短视频链接解析
- **快手视频解析** - 支持快手短视频链接解析
- **小红书视频解析** - 支持小红书视频链接解析
- **自动平台识别** - 智能识别视频来源平台

#### 2. 智能链接提取
- **自动提取链接** - 从粘贴的文本中自动提取视频链接
- **链接格式验证** - 自动验证链接格式是否正确
- **多链接处理** - 支持处理包含多个链接的文本
- **智能过滤** - 过滤无效链接，只解析有效视频链接

#### 3. 视频信息提取
- **无水印视频链接** - 提取原始无水印视频地址
- **视频封面图片** - 获取视频封面图链接
- **视频标题信息** - 提取视频标题和描述
- **视频元数据** - 获取视频时长、作者等信息

#### 4. 图集支持
- **抖音图集解析** - 支持抖音图片合集解析
- **快手图集解析** - 支持快手图片合集解析
- **批量图片展示** - 网格展示所有图片
- **图片预览功能** - 点击图片查看大图
